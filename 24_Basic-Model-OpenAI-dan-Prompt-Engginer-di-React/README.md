Kesimpulan dari pembelajaran materi Basic Model OpenAI dan Prompt Engginer di React :

1. Dengan mempelajari model dan prompt, kita dapat meningkatkan penggunaan model OpenAI, mengoptimalkan output model AI, Membangun Aplikasi AI yang lebih relevan, dan Menungkatkan daya saing di bidang AI.
2. Model OpenAI merupakan sebuah sistem AI yang dapat digunakan untuk mempelajari pola dari data dan menghasilkan output yang sesuai dengan data yang diberikan.
3. Prompt Engineer merupakan teknik yang dipakai mengoptimalkan output dari model AI dengan memberikan instruksi pada model tersebut.
